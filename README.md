# WarShip-Kampf
This is a plugin to manage simple warship fights, as MrCreativeIV invented them. 

# Commands
- /wsk
- /wsk team
- /wsk team put
- /wsk team remove
- /wsk team captain
- /wsk setup load
- /wsk setup save
- /wsk setup name
- /wsk setup spawn
- /wsk setup spectatorspawn
- /wsk setup waterheight
- /wsk setup pos1
- /wsk setup pos2
- /wsk setup world
- /wsk arena info
- /wsk arena start
